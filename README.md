# 懶惰熊 · Lazy Bear Portfolio

A Cardcaptor-Sakura-inspired personal portfolio in white & light blue.

## Stack
- Static HTML / CSS / vanilla JS — no build step.
- Deployed on Zeabur.

## Develop locally
Just open `index.html` in a browser, or run any static server:

```bash
python3 -m http.server 8080
```

## Edit content
- **Works** — `script.js`, the `works` array at the top.
- **About text & tech stack** — `index.html`, inside `.intro`.
